Energex

The superior Twike battery management system

Welcome to Energex.

This project Energex hosts a software and hardware for a Twike battery management system (BMS). Because of the Twikes DC-Link voltage of about 280V...435V a lot of cells have to be connected in series. The Energex BMS is able to charge up to 192 cells connected in series. Due to its open design it can be easily adapted to arbitrary cell type: So far NiCd, LiFePO4 and LiCoMn have been tested.

Details

The Energex project is divided in several sub projetcs:

  • [QTwikeAnalyzer], a C++ project developed with Qt is used to debug the communication between BMS and Twike converter.
  • QTwikeSimulator, also a C++ project is used to simulate charge/discharge and testing balancing algorithms.
  • [Supervisor] is a piece of hardware connected and powered directly from the battery stack. It measures the cell voltage and cell temperature and is able to drain any overcurrent from the cell.
  • [Mediator] acts as communication interface between Supervisors and Twike converter. It is responsible to adjust charge current and voltage to the state of the battery stack.
